## 🎯 What is AI DevSecOps Sentinel?

AI DevSecOps Sentinel is an AI-powered assistant that:
- Analyzes repositories for security issues
- Detects secrets and misconfigurations
- Reviews Infrastructure-as-Code files
- Provides DevOps/DevSecOps guidance
- Maps findings to compliance frameworks (CWE, OWASP, NIST)
